# Exam-invigilation-system

A modern web-based exam invigilation management system built to automate lecturer assignment, scheduling, notifications, and monitoring of examination duties. This system eliminates manual allocation errors, improves transparency, and ensures efficient communication between administrators and invigilators.

---

## Features

- Automated Exam Scheduling
- Invigilator Assignment & Reassignment
- Interactive Admin Dashboard
- Email Notifications (Mailjet Integration)
- Calendar-Based View of Duties
- Streamlit Deployment Ready

---

## Tech Stack

- Python
- Streamlit
- Mailjet API
